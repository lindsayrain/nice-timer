# NiceTimer v0.1.2

這個版本補齊 GitHub 發版流程，讓每次版本都能直接提供可下載的 macOS 安裝檔與中文說明。

## 畫面預覽

![NiceTimer 主畫面](https://raw.githubusercontent.com/lindsayrain/nice-timer/main/release-assets/screenshots/main-window.png)

## 版本重點

- GitHub Release 現在會附上 `.dmg` 與 `.zip` 兩種下載格式
- Release 頁面改用繁體中文撰寫版本說明
- Release 內容會固定帶上主畫面截圖，方便快速辨識應用程式
- README 現在直接引導使用者前往 GitHub Releases 下載穩定版本

## 下載與安裝

- 建議優先下載 `.dmg`，掛載後將 `NiceTimer.app` 拖到 `Applications`
- 若偏好手動解壓，也可下載同版本 `.zip`

## 注意事項

- 支援系統：macOS 14 以上
- 此版本為 ad-hoc 簽章，首次開啟若被 Gatekeeper 阻擋，可到「系統設定 > 隱私權與安全性」允許執行
