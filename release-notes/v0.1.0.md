# NiceTimer v0.1.0

First packaged release of NiceTimer for macOS.

## Screenshot

![NiceTimer main window](https://github.com/lindsayrain/nice-timer/blob/add-menu-bar-app/release-assets/screenshots/main-window.png?raw=1)

## Highlights

- Native macOS SwiftUI timer app
- Flip-clock main window for countdown and stopwatch modes
- Menu bar access with start, pause, reset, and countdown presets
- Shared timer state between the main window and menu bar
- Fullscreen support and keyboard shortcuts

## Included in this release

- Countdown mode with configurable hours, minutes, and seconds
- Stopwatch mode
- Quick presets for 5, 15, 25, and 45 minutes
- System alert sound when a countdown finishes
- Packaged `.app` and release zip workflow

## Notes

- Minimum supported system: macOS 14
- The packaged app is ad-hoc signed for local distribution
